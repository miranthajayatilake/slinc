Slinc

Searchable Lex Fridman Podcast

"As someone who very much enjoys podcasts I continue to be frustrated that so much information is locked up in opaque audio files." - Andrej Karpathy

But now that information is accessible and searchable on Slinc!

This app uses audio transcriptions of ~322 Lex Fridman podcasts shared by Andrej Karpathy, produced using OpenAI Whisper model. This project is open-sourced on
